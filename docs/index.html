One config, full atmosphere

Theme switching, but cinematic.

A tiny, zero-dependency JS/TS library that turns plain dark-mode into clean, weather-aware UI atmospheres. Professional by default — the cinematic layer is opt-in, and one call wires it all.

Quick start View source 
TheSwitch.create({
  defaultSkin: "midnight",   // or mode:"auto", auto:"sync"
  ambient: true,            // realistic climate effects (opt-in)
  transition: "portal",
  autoBind: true,           // wires every data-switch-* control
});

Why it stays out of your way

Clean first. Cinematic when you ask.

Three ideas keep TheSwitch professional in production and impressive on a landing page.

Clean by default

Drop it in and you get a calm, accessible light/dark system driven from :root tokens. Effects are off until you opt in.

One synced atmosphere

auto:"sync" fuses time of day + season + location + live weather into a single atmosphere — never one signal fighting another. Or auto:"system" for plain light/dark. Weather is opt-in, coarse location only.

Opt-in climate

Turn on a realistic canvas layer — sun rays, rain & ripples, snow crystals, fog, stars, ocean waves. Reduced-motion & tab-aware, never a battery hog.

Quick start

Two lines, no build step.

Use the CDN global for a plain page, or install from npm for an app. Zero runtime dependencies either way.

<!-- CDN: drop-in global -->
<script src="https://cdn.jsdelivr.net/npm/the-switch"></script>
<script>
  TheSwitch.create({ defaultSkin: "midnight", autoBind: true });
</script>
// npm: for apps & bundlers
import { createSwitch } from "the-switch";

createSwitch({
  ambient: true,        // realistic effects
  auto: "sync",      // time+season+weather+location, fused
  useGeolocation: true, // opt in to live weather
});
