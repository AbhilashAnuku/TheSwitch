One config, full atmosphere

Theme switching, but cinematic.

A tiny, zero-dependency JS/TS library that turns plain dark-mode into clean, weather-aware UI atmospheres. Professional by default; the cinematic layer is opt-in.

TheSwitch.create({
  defaultSkin: "midnight",   // or mode:"auto", auto:"time"
  ambient: true,            // realistic climate effects (opt-in)
  transition: "portal",
  autoBind: true,           // wires every data-switch-* control
});
